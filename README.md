Restaurant Data Analysis Internship Project using Python, Pandas, Matplotlib, and Seaborn with Exploratory Data Analysis and Data Visualization.



# 📊 Cognifyz Data Analysis Internship

## Overview
This project was completed as part of the Cognifyz Technologies Data Analysis Internship.

The project focuses on analyzing restaurant data and deriving meaningful insights using Python.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Tasks Completed

### Level 1
- Top Cuisines Analysis
- City Analysis
- Price Range Distribution
- Online Delivery Analysis

### Level 2
- Restaurant Ratings Analysis
- Cuisine Combination Analysis
- Geographic Analysis
- Restaurant Chain Analysis

## Key Insights
- Identified popular cuisines and restaurant trends
- Analyzed customer ratings and voting patterns
- Studied geographic distribution of restaurants
- Explored online delivery availability and pricing patterns
